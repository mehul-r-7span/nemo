# problem

We have to manage yacht rooms, guests, and menu, with a history of guests without internet connection. 

# solution 

This project involves managing yachts, decks, rooms, guests, menus, and histories using a LAN network.

For this, we have 9 different roles:
    nemo_admin
    yatch_admin
    system_admin
    chief_stewardess
    stewardess
    chef
    captain
    deck_hand
    guest

This role helps smooth the handling of yachts. Each role have thire different functionality and work in this system.
